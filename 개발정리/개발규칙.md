## 컨벤션 :triangular_ruler:

* **시작은 소문자 구분은 대문자로**

  * **ex)** takePicture

* **함수**: 동사

  * **ex)** changeUserData

* **변수**: 명사

  * **ex)** userData

* **Class:** 대문자로 시작

  * **ex)** User

* **CSS:** 태그-기능1-기능2-번호

  * **ex)** btn-cancle-01

* **JS:** const (변수, 함수) > let (중간에 값이 변하는 변수) > var (x)

* **주석:** HTML(최상단), JS, VIEWS

* **Commit:** 기능-변경파일명-날짜
